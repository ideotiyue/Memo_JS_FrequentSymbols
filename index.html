Javascriptの記号

初心者ゆえに使い分けがわからない自分へのメモ

まぎらわしい記号たち①

. (ピリオド)

階層の区切りに使う。「○○の中の〜」的なイメージ

, (コンマ)

要素を区切るときに使う。

■例
const character = {name: "わんこ", age: 4};

// characterのnameの値を出力
console.log(character.name);

まぎらわしい記号たち②

; (セミコロン)

文の最後につける。
ただし、 {}の後にセミコロンは不要（なんでや！）
for文の括弧の中で「変数の定義」「条件式」「変数の更新」の区切りにも使う。

■例
for (let number = 1; number <= 100; number++) {
console.log(number);}

: (コロン)

オブジェクト作成時、プロパティと値の間を繋ぐときに使う。
コロンの前と後に続く文が内容的にイコールと思うとセミコロンと混同しないかも。

■例
const character = {name: "にんじゃわんこ", age: 14};

まぎらわしい記号たち③

“ (ダブルクォーテーション)

文字列を入れるときに使う。
’(シングルクォーテーション)でもOKだけれど、バッククォーテーションと混同しそう。

■例
//「20」と「15」を文字列として連結
console.log("20"+"15");

` (バッククォーテーション)

テンプレートリテラルで文字列の中に定義された定数や変数を入れるときに使う。
※「${}」とセット。

■例
const age = 30;
console.log(`今は${age}歳です`);

まぎらわしい記号たち④

() (丸括弧、パーレン)

関数の引数、if文などの条件式で使う。

{} (波括弧、ブレイス)

クラス、関数、if文の本体やオブジェクトを作る場合など。
複数行にまたがるような大きめの文に使う。

[] (角括弧、ブラケット)

配列を作成するときや、値を取り出す場合に使う。

■例
const characters = [
{name: "わんこ", age: 4},
{name: “にゃんこ, age: 3}
];
// charactersの2つ目の要素の「name」に対応する値を出力
console.log(characters[1].name);
